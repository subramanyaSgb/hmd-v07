See comprehensive documentation at:

[04-statistics.md](../../developer-docs/docs/frontend/pages/04-statistics.md)

## Quick Reference

**File:** `frontend/src/pages/Statistics.jsx`

**Route:** `/statistics`

**Access:** All authenticated users

**Purpose:** Role-based analytics router that displays:
- **Admin:** System-wide analytics (AdminStatistics component)
- **Producer:** Producer-specific performance metrics (ProducerStatistics component)
- **Consumer:** Consumer-specific performance metrics (ConsumerStatistics component)
